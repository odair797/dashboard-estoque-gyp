# Pasta input/

Cole aqui o arquivo `.tsv` exportado do **Senior (Saldo de Estoque)**.

Você pode jogar a exportação diária direto — o script `atualizar.py`
pega automaticamente o `.tsv` mais recente desta pasta.

Não precisa renomear o arquivo.

(Opcional) Se tiver o arquivo de **Vendas Últimos 3 Meses** em `.xlsx`,
pode colar aqui também.
